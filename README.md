
# Todo List

A simple to-do list web application to manage your daily tasks.


## Features

- Create/Login user
- Add new tasks
- Show all tasks

## Installation

Install todo-list with npm

```bash
  Clone the repository: git clone https://github.com/MantasGulbinskas/todo-list.git
  Install dependencies: npm install
  Start the application: npm start
```
